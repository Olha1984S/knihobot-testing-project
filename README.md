## About the Project

This repository contains a comprehensive QA testing project for the public website **Knihobot.cz**.

The project demonstrates practical experience in **manual software testing**, including test planning, test case design, execution, defect reporting, and documentation. It also includes the first automated UI tests developed with **Playwright** and **Python**.

Testing was performed exclusively through the publicly accessible user interface without access to the application's source code, database, or administrative functionality.

---

## Project Scope

The following modules were tested:

- Homepage
- Website Navigation
- Search
- Filters & Sorting
- Catalog Navigation
- Product Detail Page

## Defects Identified

Testing identified several functional and UI issues, including:

- Search behavior
- Filter logic
- Product card layout
- UI consistency
- Input handling

All defects were documented with severity, reproduction steps, expected and actual results.
